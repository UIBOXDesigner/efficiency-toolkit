# Efficiency Toolkit

Chrome 效率工具插件，包含截图、笔记、格式转换等功能。

## 功能

- 📸 **截图工具** - 快速截取当前页面
- 📝 **笔记** - 快速记录和保存笔记
- 🔄 **格式转换** - JSON 格式化、Base64 编解码、时间戳转换、URL 编解码
- 📋 **任务管理** - 待办事项、智能安排任务优先级和截止日期
- 🎬 **录屏工具** - 快速录制屏幕、支持全屏/窗口/区域录制、一键分享
- 🎨 **取色器** - 网页取色、支持 HEX/RGB/HSL/HSB 格式、一键复制

## 安装

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `efficiency-toolkit` 文件夹

## 快捷键

- `Ctrl+Shift+S` / `Command+Shift+S` - 截图
- `Ctrl+Shift+N` / `Command+Shift+N` - 打开笔记

## 技术栈

- Manifest V3
- Vanilla JavaScript
- Chrome Storage API
